Sprachgesteuerte Lampe

Dieses Projekt habe ich mit einem Arduino Uno entwickelt. Die Lampe reagiert auf Sprachbefehle und schaltet eine normale Glühbirne über ein Relais an und aus. Das Projekt zeigt die Kombination von Hardware (Arduino, Breadboard, Relais, Schuko-Kabel) und Software für ein IoT-/Smart-Home-Projekt.

Verwendete Komponenten

  • Arduino Uno Controller
  • Breadboard
  • Relais-Modul
  • Schuko-Kabel
  • Normale Glühbirne

Arduino-Code

#include <Arduino.h>

// Beispielcode für Relaissteuerung
const int relaisPin = 7; // Pin, an dem das Relais angeschlossen ist

void setup() {
  pinMode(relaisPin, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  // Beispiel: Schalte Relais ein/aus über die serielle Konsole
  if (Serial.available()) {
    char command = Serial.read();
    if (command == '1') {
      digitalWrite(relaisPin, HIGH); // Lampe an
    } else if (command == '0') {
      digitalWrite(relaisPin, LOW);  // Lampe aus
    }
  }
}
